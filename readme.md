# Handwritten Digit Recognition using MATLAB

##  Project Overview

This project recognizes handwritten digits (0–9) using the MNIST dataset. It uses image processing and machine learning techniques to classify digits accurately.

##  Features

* Recognizes digits from handwritten images
* Uses MNIST dataset
* Simple and efficient classification
* MATLAB-based implementation

##  Technologies Used

* MATLAB
* MNIST Dataset
* Image Processing Toolbox

##  Project Structure

* `code.mlx` → Main implementation file
* `mnist.mat` → Dataset
* `number2.png` → Sample input image

##  How to Run

1. Open MATLAB
2. Load the project folder
3. Run `code.mlx`
4. View the output predictions

##  Output

The model predicts the digit from the given handwritten image.
